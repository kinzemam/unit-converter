# 📏 Metric / Imperial Unit Converter

A responsive and interactive unit converter built using **HTML**, **CSS**, and **JavaScript**.

This project converts values between **Metric** and **Imperial** units and also includes a **Dark Mode toggle** for improved user experience.

## ✨ Features

- Convert between Metric and Imperial units
- Length conversion (Meters ↔ Feet)
- Volume conversion (Litres ↔ Gallons)
- Mass conversion (Kilograms ↔ Pounds)
- Dark mode toggle 🌙
- Responsive and clean UI
- Interactive conversion button

## 🔢 Conversion Types

### Length
- Meters → Feet
- Feet → Meters

### Volume
- Litres → Gallons
- Gallons → Litres

### Mass
- Kilograms → Pounds
- Pounds → Kilograms
